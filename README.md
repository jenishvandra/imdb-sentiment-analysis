# 📖 Overview

This project performs **Sentiment Analysis** on IMDb movie reviews using **Natural Language Processing (NLP)** and **Machine Learning**.

The model analyzes a movie review and predicts whether the sentiment is:

- 😊 Positive
- 😞 Negative

The project demonstrates the complete ML workflow from **data preprocessing** to **model training**, **evaluation**, and **prediction**.

---

# ✨ Features

- 🎬 IMDb Movie Review Classification
- 🧹 Text Cleaning & Preprocessing

- 📊 TF-IDF Feature Extraction
- 🤖 Machine Learning Model Training

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/jenishvandra/imdb-sentiment-analysis.git
```

Move into the project folder

```bash
cd imdb-sentiment-analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python app.py
```
